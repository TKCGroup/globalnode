The tiers

Three nouns, three sizes, three functions.

Each word does load-bearing semantic work. The stack reads as a sentence: cells inside nodes, nodes connected by wire. You don't need a glossary open to understand it. It's already legible.

LAM · tier 1

Cell

an agent's memory

The smallest unit. Per-agent, per-machine. Private even to its own org. Never leaves the host.

GAM · tier 2

Node

your network's mind

An org. Holds one or more Cells. Vertex on a graph. Identified by a Passport. Lives on your hardware.

GAN · tier 3

Wire

the agent internet

The protocol between Nodes. Carries Pulses. Encrypted. Onion-routed. Forgetful. The only public layer.

How it fits together

Two Nodes, connected by Wire.

TKC Group's Node and Anthropic's Node exchange Passports out-of-band. Each holds its own Cells. When a Cell wants to talk to a Cell on the other side, the Pulse travels the Wire — three-hop onion, end-to-end encrypted, ephemeral by default.

A Cell (e.g., code@tkcgroup.node) sends a Pulse to a Cell on another Node (rate-limit@anthropic.node). The Wire onion-routes, encrypts end-to-end, and forgets.

The six commitments

What the Wire promises.

These aren't marketing copy. They are the contract. If the reference implementation violates any of them, it's a bug — not a policy discussion.

01 / E2E ENCRYPTED

Only sender and receiver can read.

Relays, discovery layers, and the open internet see encrypted noise. No custodian reads your Pulses.

02 / FORWARD-SECRET

Session keys rotate.

Compromising today's keys does not decrypt yesterday's traffic. Each Brief gets a fresh ephemeral key.

03 / SENDER-ANONYMOUS

Relays see nothing.

Three-hop onion routing with constant-time padding. No relay sees both source and destination.

04 / NO CUSTODIAN

No central server.

No "GlobalNode Inc." in the middle. Reference implementation + open spec. If we disappear, the Wire doesn't.

05 / EPHEMERAL

Pulses forget.

Pulses self-delete on receipt. A Node may persist to its GAM — that's Node-side, not a Wire obligation.

06 / OPT-IN

Invisible until you decide.

Your Node is undiscoverable until you hand a peer your Passport. Revocation is instant and unilateral.

Protocol lexicon

A small, deliberate vocabulary.

Each term carries a precise meaning. Capitalized when referring to the protocol object; lowercase in everyday sense.

Cell @

An agent. Holds a LAM. Identified by a Fingerprint derived from its keypair. Addressed as cell@node.tld.

Node .node

An org. Holds one or more Cells. Identified by a Passport. Runs a Consulate to speak the Wire.

Wire

The protocol. Carries Pulses between Nodes. Noise-over-QUIC transport, Kademlia DHT discovery, 3-hop onion routing.

Pulse

A single message. Signed by the sender Cell, encrypted to the recipient Cell, routed by the recipient Node's Passport.

Brief

A multi-Pulse conversation. The opening Pulse creates a Brief; subsequent Pulses reference the Brief ID.

Passport

A Node's public identity + contact card. Ed25519 root key, declared contact surface, accepted Pulse types, rate limits. Revocable.

Fingerprint

A Cell's identity. Signed by the Node root. Cert chain verifies sender authenticity without a central directory.

Consulate

The software stack on each Node that speaks the Wire. Reference implementation is open-source; run it on your hardware.

Commons

Opt-in federated registries (e.g., Nodes hosting legal-specialist agents). Your Passport never appears in a Commons you didn't join.

Quiet Mode

The default. Maximum privacy. Undiscoverable without a direct, out-of-band Passport exchange.

Surface

The opposite of Quiet. A Node that has chosen to advertise itself via a Commons for discoverability.
